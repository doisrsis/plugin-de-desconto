== Descrição ==  
Aplica descontos automáticos no WooCommerce com base na quantidade de itens no carrinho.  

== Instalação ==  
1. Faça upload do plugin para `/wp-content/plugins/`  
2. Ative através do painel do WordPress  
3. Configure em "Desconto Automático" no menu do WordPress
